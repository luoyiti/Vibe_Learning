# SVM · 支持向量机

**一句话总结：** 支持向量机通过最大化间隔找到最优分离超平面，借助合页损失容忍噪声、借助对偶与核技巧在隐式高维空间中处理非线性分类，最终只由少数"支持向量"决定决策边界。

---

## 1) 数学原理

### 1.1 几何间隔

样本点 $(x_i, y_i)$（$y_i \in \{+1, -1\}$）到超平面 $w^\top x + b = 0$ 的几何距离为：

$$\gamma_i = \frac{y_i(w^\top x_i + b)}{\|w\|}$$

SVM 的核心目标是**最大化最小间隔** $\max_{w,b} \frac{\min_i \gamma_i}{1}$，即让离超平面最近的样本尽可能远。

### 1.2 硬间隔 SVM（线性可分）

将"最大化间隔"等价改写为凸二次规划：

$$\min_{w,b}\ \tfrac{1}{2}\|w\|^2 \quad \text{s.t.}\quad y_i(w^\top x_i + b) \ge 1,\ \forall i$$

约束的含义：所有样本必须被正确分类，且函数间隔 $\ge 1$。最优解 $w^*$ 只由那些 $y_i(w^\top x_i+b)=1$ 的样本决定——这些就是**支持向量**。

### 1.3 软间隔与合页损失（线性不可分）

真实数据有噪声/离群点，硬间隔不可行。引入松弛变量 $\xi_i \ge 0$ 允许违反：

$$\min_{w,b,\xi}\ \tfrac{1}{2}\|w\|^2 + C\sum_i \xi_i \quad \text{s.t.}\quad y_i(w^\top x_i + b) \ge 1 - \xi_i$$

等价的合页损失形式：

$$\min_{w,b}\ \tfrac{1}{2}\|w\|^2 + C\sum_i \max(0,\, 1 - y_i(w^\top x_i + b))$$

- $C \to \infty$：退化为硬间隔（不容忍违反）
- $C \to 0$：容忍大量违反（欠拟合）
- 合页损失在 $y_i(w^\top x_i+b) \ge 1$ 时为 0，否则线性增长——"已经分对了就不再追求更好"的稀疏性来源。

### 1.4 对偶问题与 KKT

对软间隔原问题构造拉格朗日函数，由 KKT 条件得到对偶问题：

$$\max_\alpha\ \sum_i \alpha_i - \tfrac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j \langle x_i, x_j \rangle$$
$$\text{s.t.}\quad 0 \le \alpha_i \le C,\quad \sum_i \alpha_i y_i = 0$$

KKT 互补条件 $\alpha_i[y_i(w^\top x_i+b)-1+\xi_i]=0$ 揭示：
- $\alpha_i = 0$：非支持向量，对决策无贡献
- $0 < \alpha_i < C$：恰在间隔边界上的支持向量
- $\alpha_i = C$：违反间隔的边界支持向量（可能误分）

决策函数只依赖支持向量：

$$f(x) = \text{sign}\left(\sum_i \alpha_i y_i \langle x_i, x \rangle + b\right)$$

### 1.5 核技巧（非线性分类）

对偶问题中样本只以**内积** $\langle x_i, x_j \rangle$ 形式出现。用核函数 $K(x_i, x_j) = \langle \phi(x_i), \phi(x_j) \rangle$ 替换内积，即可在隐式高维特征空间中训练，无需显式计算 $\phi$：

| 核函数 | 公式 | 适用场景 |
|---|---|---|
| 线性 | $K(x,z) = x^\top z$ | 线性可分/高维稀疏 |
| RBF（高斯） | $K(x,z) = \exp(-\gamma\|x-z\|^2)$ | 通用非线性，最常用 |
| 多项式 | $K(x,z) = (x^\top z + r)^d$ | 已知特征交互阶数 |

RBF 核的 $\gamma$ 控制高斯衰减：$\gamma$ 大 → 每个样本影响范围小 → 易过拟合；$\gamma$ 小 → 决策平滑 → 易欠拟合。

## 2) 训练过程

### 2.1 线性 SVM（次梯度下降）

对合页损失形式直接做次梯度下降：

```
初始化 w = 0, b = 0
for epoch in 1..T:
    for each (x_i, y_i):
        margin = y_i * (w·x_i + b)
        if margin < 1:
            w -= lr * (w/T_reg - C * y_i * x_i)   # 次梯度
            b -= lr * (-C * y_i)
        else:
            w -= lr * (w/T_reg)                    # 仅正则项
```

### 2.2 核 SVM（简化 SMO）

SMO（Sequential Minimal Optimization）每次选两个拉格朗日乘子 $\alpha_i, \alpha_j$ 联合优化，解析求解二元子问题：

```
初始化 α = 0, b = 0
for iter in 1..max_iter:
    选违反 KKT 的 (i, j)
    解析求最优 α_i, α_j（保持 Σα_k y_k = 0 约束）
    更新 b
    剪裁 α ∈ [0, C]
until KKT 违反量 < ε
```

### 2.3 多分类（One-vs-Rest）

对 $K$ 类训练 $K$ 个二分类 SVM，第 $k$ 个把类 $k$ 作为 +1、其余作为 -1。预测时取决策值最大的类。

## 3) 效果评估

- **混淆矩阵**：直观展示各类分类情况
- **Accuracy**：整体正确率，类别均衡时有效
- **Precision / Recall / F1**：类别不均衡时更重要；macro-F1 对各类等权平均
- **K-Fold 交叉验证**：把数据分 K 份，轮流用 K-1 份训练、1 份验证，得到更稳健的性能估计

### 超参数调优

- **C**（正则参数）：大 C → 低偏差高方差；小 C → 高偏差低方差
- **γ**（RBF 核宽度）：大 γ → 高方差；小 γ → 高偏差
- 通常用网格搜索 + 交叉验证选 (C, γ)

## 4) 真实数据实战（Fisher Iris）

Iris 数据集：150 样本，4 特征（花萼/花瓣 长度/宽度），3 类。

**关键发现：**
1. **花瓣特征（petal length × petal width）几乎完美分离三类**——只需 2D 特征即可高准确率分类
2. **RBF 核 vs 线性核**：在花瓣特征上线性核已足够（数据近线性可分），RBF 核在更难的特征组合上优势明显
3. **C 的影响**：C 过大时训练集 100% 但测试集下降（过拟合）；C 适中时泛化最佳
4. **支持向量稀疏性**：通常只有 10%-30% 样本成为支持向量，模型紧凑

## 5) 要点与易错点

### 要点
- SVM 的核心是**最大化间隔**，不是最小化错误数——这赋予它更好的泛化性
- **对偶形式**是核技巧的入口：样本只以内积出现，才能用核函数替换
- **支持向量稀疏性**：决策只由少数样本决定，预测高效
- 软间隔的 $C$ 和 RBF 的 $\gamma$ 是两个最关键的超参数

### 易错点
- **不缩放特征**就训练：SVM 对特征尺度敏感（间隔依赖距离），必须先标准化
- **用 Accuracy 评估不均衡数据**：应看 macro-F1 / 混淆矩阵
- **混淆 C 的方向**：大 C = "更严格" = 低偏差高方差（不是正则更强）
- **γ 过大**：每个支持向量只影响极小邻域，决策边界碎片化 → 严重过拟合
- **核 SVM 预测慢**：需对所有支持向量求核，$O(n_{sv} \cdot d)$；大数据集考虑近似方法

---

## 6) 与其他模型的关系

- **vs 逻辑回归**：都是线性分类器，但损失函数不同（合页 vs 对数似然）；SVM 间隔最大化带来稀疏解，LR 概率输出更自然
- **vs 决策树**：决策树轴对齐递归划分，SVM 在（核映射后的）高维空间找最大间隔超平面；树天然可解释，SVM 更稳健
- **One-Class SVM / SVDD**：SVM 思想推广到异常检测（只用正类训练，学一个把正类包住的最小球）
