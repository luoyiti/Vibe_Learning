# 逻辑回归 (Logistic Regression)

**一句话：** 逻辑回归在线性回归的打分外面套一个 Sigmoid，把实数打分压成 (0,1) 的概率，
再用极大似然（交叉熵损失）学出一条线性决策边界——名字带"回归"，本质是**概率分类器**。

## 1. 数学推理

**模型。** 先做线性打分 `z = w·x + b`，再过 Sigmoid 得到正类概率：

```
P(y=1 | x) = σ(z) = 1 / (1 + e^(-z))
```

等价的关键视角：**对数几率（log-odds / logit）是 x 的线性函数**——这才是"线性"所在：

```
log( P(y=1) / P(y=0) ) = w·x + b
```

**损失。** 把每个样本视作伯努利试验，对似然取负对数 → **二元交叉熵**（凸函数）：

```
L(w) = -(1/n) Σ [ y_i·log(p_i) + (1-y_i)·log(1-p_i) ],   p_i = σ(w·x_i + b)
```

为什么不用平方误差？套上 Sigmoid 后平方误差非凸、多局部极小；交叉熵对 Sigmoid 是凸的，
梯度下降保证全局最优。

**梯度。** 求导后 Sigmoid 导数项约掉，化简成极简形式：

```
∂L/∂w = (1/n) Σ (p_i - y_i)·x_i
```

梯度 = "预测概率与真实标签之差 × 特征"，错得越多推得越狠。

## 2. 训练方式

更新规则：`w ← w - η·(1/n) Σ (p_i - y_i) x_i`。

- **批量 GD**：用全部样本，稳但慢。**SGD / mini-batch**：大数据首选。
- **牛顿法 / IRLS**：用二阶 Hessian，几步收敛，统计软件常用。**L-BFGS** 是 sklearn 默认。
- **正则化（必备）**：L2(Ridge) 加 `λ‖w‖²` 让系数变小；L1(Lasso) 加 `λ‖w‖₁` 把不重要系数压到 0
  做特征选择；Elastic Net 折中。sklearn 用 `C = 1/λ`，C 越小正则越强。
- **前提**：特征标准化（均值0方差1）能显著加速收敛、让系数可比、让正则公平。

**正则化路径 (regularization path)。** 正则不是开关而是连续强度轴。把 λ 从小到大扫一遍、
画出每个系数随 λ 变化的轨迹，就是正则化路径：

- **L2 / Ridge** `+λ‖w‖²`：圆形等高线，系数平滑收缩、几乎不精确归零；共线性下更稳。
  近端更新 `w ← w/(1+ηλ)` 对任意 λ 数值稳定（显式 `w←w−ηλw` 在 λ 大时会发散）。
- **L1 / Lasso** `+λ‖w‖₁`：菱形尖角落在坐标轴 → 部分系数**恰好为 0** = 自动特征选择，
  近端更新是软阈值 `w ← sign(w)·max(|w|−ηλ, 0)`；λ 越大、非零系数越少。
- **选 λ**：交叉验证（`LogisticRegressionCV` / `glmnet` 的 `lambda.1se`）。强度也写作 `C=1/λ`。

## 3. 训练过程

权重从 0 开始，每一步：算预测概率 → 求与标签的差（梯度方向）→ 按学习率 η 推权重一点点 →
决策边界 `w0 + w1·x1 + w2·x2 = 0` 逐渐被数据推到正确位置，损失单调下降。
η 太大震荡发散，太小学得极慢。（交互页有动画演示决策边界移动 + 损失曲线。）

## 4. 真实数据实战

经典数据集：20 名学生"复习时长 vs 是否通过"。现场跑梯度下降拟合出 S 曲线，
得到"学 x 小时通过的概率"，以及 50% 临界时长 `-b/w`。每多复习 1 小时，几率 odds 变 `e^w` 倍。

## 5. 评估方式

模型输出概率，需**选阈值**才能下判断（默认 0.5）。

- **混淆矩阵** → 准确率/精确率/召回率/F1。阈值↑ → 精确率↑、召回率↓（此消彼长）。
- **ROC 曲线 / AUC**：扫所有阈值画"真正例率 vs 假正例率"，AUC 衡量排序能力、**与阈值无关**，
  0.5=随机，1=完美。
- 坑：**类别不平衡时准确率会骗人**，要看 P/R/F1/AUC。场景决定取舍——医疗筛查重召回，垃圾邮件重精确。

## 6. 可解释性

最大卖点：**系数可直接读懂**。`w` 作用在对数几率上，取指数 `e^w` 得到**几率比 (odds ratio)**：
正系数=危险因素，负系数=保护因素，0=无关。这种透明度让它在医疗、金融风控、信用评分仍是首选
——监管要求"能解释为什么拒绝"。

## 7. 最新发展

- **Softmax / 多项逻辑回归**：多分类推广，正是神经网络最后一层做的事。
- **GAM 广义可加模型**：线性项换平滑函数，保留可解释性又添非线性。
- **概率校准**：Platt scaling 本质就是用逻辑回归给别的模型输出校准概率。
- **为什么 2026 年还在用**：强基线（深度模型对照线）；可解释/合规刚需；深度学习的积木
  （每个二分类神经元 = 一个逻辑回归，输出层 softmax = 多项逻辑回归）；GLM 家族做统计推断；
  CTR 预估等大规模在线学习（配合 FTRL）。

## 要点 / 易错点

- 它是**判别式**模型，直接建模 P(y|x)，决策边界是**线性**超平面（线性不可分需手动加特征/核）。
- 交叉熵凸 → 必收敛全局最优；用平方误差会非凸，别用。
- 一定要**标准化特征**再训练，否则收敛慢且正则不公平。
- 评估别只看 accuracy；不平衡数据看 AUC / F1 / PR 曲线。
- 系数解释的是**对数几率**的变化，要 `e^w` 才是几率倍数——别直接当概率读。
