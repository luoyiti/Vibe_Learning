物理层定义数据作为信号在信道上发送时的电气、时序与接口特性，是协议栈的根基；介质特性与奈奎斯特/香农极限共同划定了任何信道的理论上界。

## 传输介质

**导向介质**沿物理路径传输：双绞线（差分、Cat 5e–8）、同轴电缆（50Ω/75Ω）、光纤（全内反射，单模/多模，三波段 0.85/1.30/1.55 μm）、电力线（数据叠加电力信号）。海量离线传输可用磁带/硬盘运输（Snowmobile 可达 100 PB），有效带宽远超网络。

**非导向介质**通过电磁波：无线电（全向、路径损耗 1/r²）、微波（定向、视距、多径衰落）、红外（短距）、卫星（GEO/MEO/LEO）。扩频技术包括跳频（蓝牙）、直序 CDMA（3G/GPS）、UWB。

光纤相对铜线：更高带宽、更低衰减（~50 km/中继 vs ~5 km）、抗 EMI、更安全；但成本与技能要求更高。

## 理论基础

傅里叶分析表明有限带宽信道滤掉高频谐波，限制可重构的比特速率。"带宽"有两义：Hz（模拟）与 b/s（数字）。

- **奈奎斯特**（无噪声）：最大速率 = 2B log₂V
- **香农**（有噪声）：最大速率 = B log₂(1 + S/N)

例：ADSL ~1 MHz + 40 dB SNR → 上限约 13 Mb/s，无法通过调制技巧突破。

## 数字调制与多路复用

**基带**（有线）：NRZ、NRZI、曼彻斯特（自带时钟）、4B/5B（打破长 0 串）。

**通带**（无线/光纤）：ASK、FSK、PSK、QAM——调节载波幅/频/相。

**多路复用**：FDM/OFDM（频分）、TDM（时分）、CDMA（正交码片）、WDM/DWDM（光域频分）。OFDM 用于 802.11/4G/ADSL；CDMA 用 Walsh 码正交内积解码。

## 三大通信系统

1. **PSTN**：本地回路（ADSL/FTTx）+ SONET/WDM 骨干 + 电路/分组交换
2. **蜂窝**：1G 模拟 → 2G GSM → 3G CDMA → 4G 全 IP → 5G MIMO/小蜂窝
3. **有线电视**：HFC + DOCSIS，节点共享带宽

频谱分配受政策严格管制，与技术同样塑造物理层。

## 易错点

- 带宽 Hz 与 b/s 不可混用
- 香农容量是硬上限，非"建议值"
- 有线电视/DOCSIS 带宽在节点用户间共享，非独享
- TDM ≠ STDM（后者即统计复用/分组交换）
