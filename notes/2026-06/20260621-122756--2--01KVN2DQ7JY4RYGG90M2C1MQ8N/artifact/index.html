第2章 物理层

计算机网络 · Physical Layer

物理层定义数据作为信号在信道上发送时的电气、时序与接口特性，是所有网络协议栈的根基。 介质特性决定吞吐量、延迟与误码率；奈奎斯特与香农极限则为任何信道划定了理论上界。

2.0 章节概览

物理层关注比特如何在真实世界中以信号形式传输。本章从三种传输路径展开：导向介质（铜线、光纤）、无线（地面无线电）与卫星；继而分析信道容量极限、数字调制与多路复用；最后考察电话网、蜂窝网与有线电视三大广域接入基础设施。

2.1–2.3 传输介质与频谱

导向介质

  • 磁带/存储运输：海量数据场景下有效带宽可达 Tb/s 级（"Snowmobile"）
  • 双绞线：Cat 5e/6/7/8，差分传输抗噪；全/半/单工
  • 同轴电缆：50Ω 数字 / 75Ω 有线电视，带宽可达 6 GHz
  • 电力线：数据叠加在 50–60 Hz 电力信号上，家庭短距可达 500 Mb/s
  • 光纤：全内反射；多模 vs 单模；0.85/1.30/1.55 μm 三波段；WDM 潜力 >50 Tb/s

非导向介质

  • 无线电：全向传播，路径损耗 ∝ 1/r²；VLF–MF 沿地，HF–VHF 电离层反射
  • 微波：定向、需视距中继；多径衰落；>10 GHz 受雨水吸收
  • 红外：短距、不可穿墙，无需频谱许可
  • 扩频：跳频（蓝牙）、直序 CDMA（3G/GPS）、UWB（>500 MHz 带宽）
介质优势局限
光纤 vs 铜线更高带宽、更低衰减（~50 km/中继）、抗 EMI、更安全成本高、需专业技能、单向需双纤或双波段
微波 vs 光纤无需路权、部署快、相对便宜需视距、受天气影响、容量低于光纤

2.4.1–2.4.2 理论基础

傅里叶分析：任何周期信号可分解为正弦/余弦谐波之和。有限带宽信道会滤掉高频分量，导致信号失真——带宽越窄，可承载的数据速率越低。注意"带宽"有两义：电气工程师指 Hz，计算机科学家指 b/s。

奈奎斯特（无噪声）：最大速率 = 2B log₂V
B = 信道带宽(Hz)，V = 信号离散等级数；二进制 3 kHz 信道上限 6000 b/s
香农（有噪声）：最大速率 = B log₂(1 + S/N)
S/N 为信噪比；ADSL ~1 MHz + 40 dB SNR → 上限约 13 Mb/s，无法逾越

这两个极限是物理层给所有信道施加的根本约束：无噪声看奈奎斯特，有噪声看香农。

2.4.3 数字调制

信道承载模拟信号，比特须通过数字调制映射到物理量上。

基带传输（有线常见）

  • NRZ：正/负电压或光/无光表示 1/0
  • NRZI：跳变表 1（USB 采用）
  • 曼彻斯特：每比特中间跳变，自带时钟，带宽翻倍（经典以太网）
  • 4B/5B / 8B/10B：打破长 0 串，兼顾时钟恢复与 DC 平衡

通带传输（无线/光纤常见）

  • ASK：幅移键控
  • FSK：频移键控
  • PSK：BPSK / QPSK（相位偏移）
  • QAM：幅相联合，每符号多位（如 QAM-64）

符号率（波特率）× 每符号比特数 = 比特率。多级信号可在不增加带宽的情况下提高速率，但需足够 SNR 区分各级。

2.4.4 多路复用

技术原理典型应用
FDM频谱分带，每用户独占频段 + 保护带AM 广播、早期电话
OFDM正交子载波紧密排列，FFT 实现；抗衰落802.11、4G、ADSL、电力线
TDM时间轮转，每用户固定时隙电话网、蜂窝网
CDMA正交码片序列，全频段同时发送3G、GPS、有线电视
WDM/DWDM光域 FDM，不同波长并行光纤骨干，单纤 >16 Tb/s

CDMA 类比：各站"说不同语言"——接收方用码片内积提取目标信号，其余视为噪声。Walsh 码保证正交性。

2.5–2.10 通信系统与政策

公共电话交换网 (PSTN)

  • 结构：本地回路 → 中继线 → 交换机
  • ADSL：OFDM 子载波，上行 ~3.5 Mb/s，下行 ~40 Mb/s
  • FTTx/PON：光纤到户，速率高于 ADSL
  • SONET/SDH + WDM 骨干；T1/T3 时分复用
  • 电路交换 vs 分组交换并存

蜂窝网络演进

  • 1G：模拟 AMPS
  • 2G：数字 GSM（全球最广）
  • 3G：宽带 CDMA
  • 4G：全 IP 分组交换核心，语音也走数据包
  • 5G：更小蜂窝、Massive MIMO、更多频谱

有线电视 (HFC)

  • 混合光纤同轴：光纤到节点，同轴到户
  • DOCSIS 标准；上行 TDM + 下行广播
  • 容量由服务节点用户共享（~1 Gb/s，未来 10 Gb/s）

卫星与政策

  • GEO（同步）、MEO、LEO（Starlink 等）
  • GEO 延迟高；LEO 低延迟但需大量卫星
  • 频谱分配受监管机构严格管制，拍卖与再分配持续进行

2.11 本章要点

物理层为所有上层协议划定边界：介质决定可达带宽与误码特性，奈奎斯特与香农给出理论上限； 调制将比特映射到模拟信号，复用让单条链路服务多路会话；电话网、蜂窝网与有线电视是现实中 "最后一公里"与骨干的三条主路径，其演进方向是从电路交换走向分组交换、从铜线走向光纤、 从模拟走向全数字——而频谱政策则与技术同样深刻地塑造着这一层。
